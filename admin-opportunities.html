Sync from Google Sheets

Open Google Sheets
How it works: Manage all your opportunities in a Google Sheet — add rows, edit details, mark deadlines. Click Sync here and the listings update instantly on Afrorama. No CSV export needed.

Step 1 — Set up your Google Sheet

Create a sheet with these exact column headers in row 1 (copy and paste):

title  |  organisation  |  type  |  sector  |  location  |  country  |  deadline  |  salary  |  apply_url  |  description  |  requirements  |  paid_listing
type: jobs · internship · consultancy · capacity    country: ISO 2-letter code e.g. KE ZA NG    deadline: YYYY-MM-DD    paid_listing: TRUE or FALSE

Step 2 — Add the Apps Script

In your Google Sheet: Extensions → Apps Script. Paste this code, click Save, then Deploy → New deployment → Web app → Access: Anyone → Deploy. Copy the URL.

function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const rows = sheet.getDataRange().getValues(); const keys = rows[0].map(k => k.toString().toLowerCase().trim().replace(/\s+/g,'_')); const data = rows.slice(1).filter(r => r[0]).map((r, i) => { const obj = {}; keys.forEach((k, j) => { obj[k] = r[j] !== undefined && r[j] !== null ? r[j] : ''; }); // Format date fields if (obj.deadline instanceof Date) obj.deadline = obj.deadline.toISOString().split('T')[0]; if (typeof obj.deadline === 'number') obj.deadline = new Date((obj.deadline - 25569) * 86400000).toISOString().split('T')[0]; obj.paid_listing = String(obj.paid_listing).toLowerCase() === 'true'; obj.id = obj.id || ('sheets-' + (i + 1)); obj.posted = obj.posted || new Date().toISOString().split('T')[0]; return obj; }); return ContentService .createTextOutput(JSON.stringify({ data, updated: new Date().toISOString() })) .setMimeType(ContentService.MimeType.JSON); }

Step 3 — Sync

Paste your Apps Script web app URL below. The URL is saved in your browser so you only need to do this once.

Preview

Auto-sync tip: Bookmark this admin page and click Sync once a day to keep the site current with your Sheet. UTM tracking & Bitly links

How UTM tracking works: Every "Apply Now" link includes UTM parameters so you can see in Google Analytics which listings drive applications.
utm_source=afrorama · utm_medium=jobboard · utm_campaign=[organisation] · utm_content=[job-title]

Bitly links are generated via the Bitly API and stored with each listing. They wrap the UTM URL into a short, trackable link. Set your BITLY_TOKEN in Supabase secrets and deploy the Edge Function to enable auto-generation.

Generate UTM link manually

Bitly API setup

# 1. Get a free Bitly account at bitly.com # 2. Go to Settings → Developer → API → Generate token # 3. Store in Supabase secrets: supabase secrets set BITLY_TOKEN=your_token_here # 4. Deploy the Edge Function: supabase functions deploy bitly-link # 5. The function auto-generates short links when you post a listing. # Test manually: curl -X POST https://YOUR_PROJECT.supabase.co/functions/v1/bitly-link \ -H "Authorization: Bearer YOUR_ANON_KEY" \ -H "Content-Type: application/json" \ -d '{"long_url":"https://afrorama.org/opportunities.html?id=seed-1&utm_source=afrorama&utm_medium=jobboard&utm_campaign=UNDP"}'
